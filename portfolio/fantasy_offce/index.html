이세계 동사무소

소개

이세계의 동사무소를 운영하는 액션 퍼즐게임(2D)

팀 구성

  • 6인으로 구성
  • 팀장 1명, 기획자 2명, 프로그래머 2명, 아트 1명
  • 프로그래머로 참여

기술

Unity(C#), Python 3.x, Json, csv

설명

  • Unity 엔진을 이용해 개발하고 있습니다.
  • 기획자들과 협업을 위해 대사 스크립트나 케릭터 정보들을 csv로 받고 json으로 변환시키는 python 스크립트를 작성하였습니다.
